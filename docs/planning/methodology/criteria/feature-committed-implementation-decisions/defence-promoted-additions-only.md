# Entries Are Added Via Defence Promotion

New entries and amendments enter the artifact only by promotion from a
Code Contract Defence's Committed-Decision Promotion Draft. Direct edits
bypass the only record of which task settled each fact and why.

An entry without a source promotion cannot be defended later.

## Illustrations

**Bad — direct add.** `cid-9 — All services emit metrics` is added between
tasks because it seems obvious.
**Good:** The next task touching a service drafts `cid-9` in its Code
Contract Defence promotion draft; the entry lands when the defence is
persisted.

**Bad — silent amendment.** `cid-3`'s rationale is edited for clarity with
no source-task link.
**Good:** A subsequent task amends `cid-3` in its promotion draft; the
amendment lands with `Source: t-<n>` and an `Original: cid-3` reference.
