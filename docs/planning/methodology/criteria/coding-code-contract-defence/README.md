# Code Contract Defence Criteria Catalog

Reusable criteria for the Code Contract Defence — the per-task
evidence-backed proof that the assigned slice of the feature-level Code
Contract and the task-level Coding Code Contract are satisfied.

This catalog is a library. A criterion has no authority until a Code
Contract Defence selects it. Catalog authoring follows the rules in
[../README.md](../README.md).

## How to use

A Code Contract Defence lives at
`docs/plans/<feature>/tasks/t-<n>/code-contract-defence.md`. To assemble:

1. Browse this catalog — read each criterion's `when` field
2. Select criteria that apply
3. Add an **Applicable Criteria** table near the top of the defence with
   one-line rationale per selection

See [creating-a-coding-code-contract-defence](../../creating-a-coding-code-contract-defence.md)
for full assembly guidance.
